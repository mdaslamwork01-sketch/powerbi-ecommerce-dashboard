# 📊 E-commerce Sales Dashboard (Power BI)

## 🚀 Project Overview
This project is an interactive Power BI dashboard built to analyze e-commerce sales performance.

## 📌 Key Features
- KPI tracking (Sales, Profit, Quantity, AOV)
- Monthly profit analysis (profit vs loss)
- Customer and state-wise insights
- Category & sub-category performance
- Interactive filters (Quarter slicer)

## 🛠 Tools Used
- Power BI
- Data Visualization
- Data Modeling
